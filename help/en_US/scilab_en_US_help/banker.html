banker

Banker's Algorithm

Syntax

Available = banker(Alloc,Max,Available,p,r)

Arguments

Alloc

Matrix of size (pxr) that defines the number of resources of each type currently allocated to each process.

Max

Matrix of size (pxr) that defines the maximum demand of each process in a system.

Available

Vector of size (1xr) that defines the number of available resources of each type.

p

Total Number of processes in the system

r

Total Number of resources in the system

Description

This function predicts whether the system is in safe or unsafe state based on the process execution state and also returns the Available vector after each process execution

Examples

Available = banker([3,0,1,4;2,2,1,0;3,1,2,1;0,5,1,0;4,2,1,2],[4,2,1,2;5,2,5,2;2,3,1,6;1,4,2,4;3,6,6,5],[1,5,2,0],5,4)
Available = banker([0,0,1,2;1,0,0,0;1,3,5,4;0,6,3,2;0,0,1,4], [0,0,1,2;1,7,5,0;2,3,5,6;0,6,5,2;0,6,5,6],[1,5,2,0],5,4)
