fcfs

First come first serve scheduling algorithm.

Syntax

a = fcfs(num,btime,p)

Arguments

num

Number of processes.

btime

The array containing the burst time of processes.

p

The value of this determines whether to print the output or not(1/0).

The default (optional parameter) value of this is 0.

If the value is 0 it will only return otherwise it will print and return.

a

It return an array containing the turn around time of the processes.

Description

This function takes in an array of burst time of processes and return the turnaround time of each process as an array through FCFS algorithm.

Examples

a=fcfs(4,[10,2,8,6])
a=fcfs(4,[10,2,8,6],1);
