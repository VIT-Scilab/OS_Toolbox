firstfit

First Fit partition allocation algorithm.

Syntax

[block_no, block_size] = firstfit(process, block, p)

Arguments

process

The array containing the size of each process.

block

The array containing the size of each block.

p

The value of this determines whether to print the output or not(1/0).

The default (optional parameter) value of this is 0.

If the value is 0 it will only return otherwise it will print and return.

block_no

The array containing the positions of the allocated blocks, where the index of the array refers to the process number and the value of the array refers to the block position or number. If the value stored in the array is -1, it indicates that no block could be allocated for the process.

block_size

The array containing the sizes of the allocated blocks, where the index of the array refers to the process number and the value of the array refers to the block size. If the value stored in the array is -1, it indicates that no block could be allocated for the process.

Description

This function takes in an array of process sizes and block sizes and returns the positions and sizes of the blocks allocated according to the First Fit partition allocation algorithm.

Examples

[block_no, block_size]  = firstfit([90,20,50,200],[50,100,90,200,60], 1);
[block_no, block_size]  = firstfit([90,20,50,200],[50,100,90,200,60], 0);
