look

LOOK Disk Scheduling Algorithm.

Syntax

[tot_seek, seq] = look(head, ar, p)

Arguments

head

The initial position of the disk head.

ar

This array represents the request array containing the disk track numbers.

p

The value of this determines whether to print the output or not(1/0).

The default (optional parameter) value of this is 0.

If the value is 0 it will only return otherwise it will print and return.

tot_seek

The total seek count.

seq

The array containing the seek sequence, the sequence in which requests are serviced.

Description

This function takes in an array of disk track numbers and returns the total seek count and seek sequence according to the LOOK Disk scheduling algorithm.

Examples

[tot_seek, seq] = look(50, [176, 79, 34, 60,92, 11, 41, 114 ], 1);
[tot_seek, seq] = look(50, [176, 79, 34, 60,92, 11, 41, 114 ], 0);
