lru

Least recently used page replacement algorithm.

Syntax

[faults] = lru(no_frames, no_pages, ref_string, p)

Arguments

no_frames

Number of frames.

no_pages

Number of pages.

ref_string

Page reference string in an array format.

p

The value of this determines whether to print the output or not(1/0).

The default (optional parameter) value of this is 0.

If the value is 0 it will only return otherwise it will print and return.

faults

Number of page faults.

Description

This function takes in the number of frames, number of pages and the reference string and returns the number of page faults using least recently used page replacement algorithm.

Examples

[faults] = lru(3, 7, [1, 3, 0, 3, 5, 6, 3], 1)
[faults] = lru(3, 7, [1, 3, 0, 3, 5, 6, 3], 0)
