npriority

Non premptive priority scheduling.

Syntax

[a,b] = npriority(n,priority,at,bt,p)

Arguments

n

Number of processes.

priority

Priority queue as an array format.

at

Arrival time of processes as an array.

bt

Burst time of processes as an array.

p

The value of this determines whether to print the output or not(1/0).

The default (optional parameter) value of this is 0.

if the value is 0 it will only retun otherwise it will print and return.

a

Turn around time for the processes as an array.

b

Process id in the order of execution as an array.

Description

This function takes in arrival time,burst time and priority of different processes and returns the turnaround time and proccess id through non-premptive priority scheduling algorithm.

Examples

//Only returns turnaround time
tat=npriority(5,[5,3,1,2,4],[3,2,15,1,10],[20,34,12,14,24])
//Returns the turn around time and prints the output
tat=npriority(5,[5,3,1,2,4],[3,2,15,1,10],[20,34,12,14,24],1)
//Returns the turn around time and process id and prints the output
[tat,p]=npriority(5,[5,3,1,2,4],[3,2,15,1,10],[20,34,12,14,24],1)
