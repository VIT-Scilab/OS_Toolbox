rr

Round Robin Scheduling Algorithm

Syntax

[sum_turnaround,sum_wait] = rr(at,bt,n,time_quantum)

Arguments

at

Arrival time of 'n' processes in a vector of size (1xn)

bt

Burst time of 'n' processes in a vector of size (1xn)

n

Number of processes

time_quantum

The threshold time limit for each process execution

sum_turnaround

The total turnaround time of 'n' processes in order.

sum_wait

The total waiting time of 'n' processes in order.

Description

This function takes in the number of processes, time quantum for each process,arrival time,burst time and returns the total waiting time and total turnaround time using Round Robin scheduling algorithm.

Examples

[sum_turnaround,sum_wait] = rr([0,1,2,3],[5,4,2,1],4,2)
[sum_turnaround,sum_wait] = rr([0,1,2,3,4],[5,3,1,2,3],5,2)
