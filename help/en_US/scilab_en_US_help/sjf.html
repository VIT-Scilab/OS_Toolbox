sjf

Shortest job first scheduling algorithm.

Syntax

[turnaround] = sjf(num,pt,p)

Arguments

num

Number of processes.

pt

The burst time of 'n' processes in an array format.

p

The value of this determines whether to print the output or not(1/0).

The default (optional parameter) value of this is 0.

If the value is 0 it will only return otherwise it will print and return.

turnaround

The turnaround time of 'n' processes in order.

Description

This function takes in the number of processes and burst time and returns the waiting time and turnaround time as an array using shortest job first algorithm.

Examples

[turnaround] = sjf(5, [4, 1, 5, 6, 2], 1)
[turnaround] = sjf(5, [4, 1, 5, 6, 2])
