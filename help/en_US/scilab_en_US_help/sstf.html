sstf

Shortest seek time first Disk scheduling algorithm.

Syntax

[a,b] = sstf(ar,initial_pos,p,g)

Arguments

ar

Seek positions in an array format.

initial_pos

The starting position of the head.

p

The value of this determines whether to print the output or not(1/0).

The default (optional parameter) value of this is 0.

if the value is 0 it will only retun otherwise it will print and return.

g

The value of this determines whether to display the graph or not(1/0).

The default (optional parameter) value of this is 0.

if the value is 0 it won't display the comparison graph.

a

Seek sequence.

b

Total seek count.

Description

This function takes in different seek positions as an array and reutrn the seek sequence and total seek count using shortest seek time first algorithm.

It also displays the graphical comparison of ffcs seek vs sstf seek for the given seek positions.

Examples

//Only returns the seek sequence
a=sstf([52,62,120,114,20],45)
//Returns the seek sequence and prints the output
a=sstf([52,62,120,114,20],45,1)
//Returns the seek sequence and seek count and displays the comparison graph
[a,s]=sstf([52,62,120,114,20],45,g=1)
//Returns the seek sequence and seek count,prints the output and also displays the comparison graph
[a,s]=sstf([52,62,120,114,20],45,1,1)
